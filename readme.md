# U校园全自动助手 - 项目说明



这是一个为 **U校园ai版 新一代大学英语 发展篇(unipus.cn)** 设计的Tampermonkey（油猴）脚本，旨在通过自动化处理课程任务，帮助用户节省时间，专注于更重要的学习内容。脚本的设计初衷是希望可以帮助大学生免受u校园这个非人类的东西的困扰，我买激活码买了两年，都给淘宝送百多块了，还不包括书本费，这玩意儿是真恶心


## 🚀 功能特性

*   **全自动任务执行**: 启动后，脚本将严格按照预设的“动作序列”自动完成单元内的所有任务。
*   **通用单元支持**: 一套逻辑模板即可支持全部单元（1-8），无需为每个单元编写不同代码。
*   **智能单词卡模块**:


*   **复杂流程处理**: 独创的“动作序列”引擎，能够完美处理如“单元测试Part III”中“答题 → 下一题 → 答题 → 提交”等复杂的多步操作。
*   **智能提交与导航**: 自动处理“提交”、“下一题”以及“继续学习”等按钮，实现任务间的无缝切换。



## 🔧 安装与使用

1.  **安装用户脚本管理器**:
    你的浏览器需要一个用户脚本管理器才能运行此脚本。推荐使用 **Tampermonkey**。
    *   [点此为你的浏览器安装Tampermonkey](https://www.tampermonkey.net/)



2.  **使用方法**:
    *   安装成功后，打开U校园的课程页面 (`*://ucontent.unipus.cn/*`)。
    *   脚本的操作面板将自动出现在页面左上角。
    *   从下拉菜单中选择你的课程对应的题库。
    *   点击 **“▶️ 开始自动学习”** 按钮，脚本将开始工作。
    *   如需暂停，随时可以点击 **“⏹️ 停止自动化”**。

## 🤝 如何贡献：添加新课程题库

我们非常欢迎你为项目贡献新的课程题库，让脚本能帮助更多人。过程非常简单：

#### 第1步: Fork本项目
在GitHub上，点击本项目右上角的 **Fork** 按钮，将项目复制一份到你自己的仓库中。

#### 第2步: 创建你的JSON题库文件
1.  参照项目中的 `U校园 新一代大学英语 综合教程1英语题库.json` 文件格式，创建一个新的 `.json` 文件。
2.  文件结构必须遵循以下格式：
    ```json
    {
      "Unit 1": {
        "iExplore 1: Learning before class": {
          "Reading comprehension": "答案...",
          "Dealing with vocabulary": "答案..."
        },
        "Unit test": {
          "Part I": "答案...",
          "Part II": "答案...",
          "Part III": "翻译或作文答案..."
        }
      },
      "Unit 2": {
        "...": "..."
      }
    }
    ```

#### 第3步: 托管JSON文件并获取Raw链接
你需要将你的JSON文件上传到一个公开的、能直接访问原始文件的地方。**强烈推荐使用GitHub**。
1.  在你的GitHub上创建一个新的**公开**仓库（或使用现有仓库）。
2.  将你创建的 `.json` 文件上传到这个仓库中。
3.  在GitHub上点开你上传的JSON文件，然后找到并点击 **“Raw”** 按钮。
4.  浏览器会跳转到一个只包含纯文本内容的新页面，**复制这个页面的URL**。这个就是我们需要的“Raw链接”。
    *   **错误链接**: `https://github.com/user/repo/blob/main/my-answers.json`
    *   **正确链接**: `https://raw.githubusercontent.com/user/repo/main/my-answers.json`

#### 第4步: 在脚本中添加你的题库信息
1.  在你的Fork的项目中，编辑主脚本文件 (`.user.js`)。
2.  找到 `COURSE_MAPS` 常量，在其中添加一行，指向你的题库。
    ```javascript
    const COURSE_MAPS = {
        '新一代大学英语 综合教程1': 'https://raw.githubusercontent.com/twj0/Unipus_AI-4/main/U%E6%A0%A1%E5%9B%AD%20%E6%96%B0%E4%B8%80%E4%BB%A3%E5%A4%A7%E5%AD%A6%E8%8B%B1%E8%AF%AD%20%E7%BB%BC%E5%90%88%E6%95%99%E7%A8%8B1%E8%8B%B1%E8%AF%AD%E9%A2%98%E5%BA%93.json',
        // 在下面添加你的课程
        '你的课程完整名称': '你获取的Raw链接'
    };
    ```

#### 第5步: 提交Pull Request
保存你的修改，然后向本项目的原始仓库提交一个 **Pull Request**。一旦审核通过，你的贡献就会合并到主脚本中！非常感谢！

## 💡 油猴使用技巧

*   **强制更新脚本**: 在油猴的管理面板中，可以手动点击“检查更新”来获取最新版本的脚本。
*   **查看日志**: 如果脚本运行不正常，可以按 `F12` 打开浏览器的“开发者工具”，切换到 **“控制台” (Console)** 选项卡，脚本的所有操作日志都会在这里打印出来，非常便于排查问题。
*   **编辑脚本**: 在油猴的管理面板中，点击脚本的编辑按钮（一支笔的图标），可以查看和修改脚本的源代码。
*   **备份**: 在对脚本进行任何重大修改之前，建议先将原始代码复制到文本文件中进行备份。
